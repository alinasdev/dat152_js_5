Serialize instance of Class

All output is to the web console only.

Click the button to serialize instance of the following class: 

class Student {
    // Using "_" to indicate a "private" property
    _name = "Ole Olsen";

    constructor(name) {
        this._name = name;
    }

    get name() { return this._name }

    set name(name) { this._name = name }

    getName() {
        return this._name;
    }
}
