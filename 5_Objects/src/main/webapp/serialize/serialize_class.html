Serialize instance of Class

All output is to the web console only.

Click the button to serialize instance of the following class: 

class Student {
    #name = "Ole Olsen";
    phone = "12345678";

    constructor(name,phone) {
        this.#name = name;
        this.phone = phone;
    }

    get name() { return this.#name }
    set name(name) { this.#name = name }
    
    getName() {
        return this.#name;
    }
    
    getPhone() {
        return this.phone;
    }
}
